## ШАГ 2 — Задеплой на Vercel

1. Зайди на https://vercel.com и войди через GitHub
2. Нажми **"Add New Project"**
3. Найди репозиторий **bobur-photo** и нажми **"Import"**
4. В разделе **"Environment Variables"** добавь эти переменные:

   | Имя | Значение |
   |-----|----------|
   | IMGBB_API_KEY | 8093b5a4acf05371a044d92054ea6cd0 |
   | ADMIN_PASSWORD | foto2025 |
   | NEXT_PUBLIC_SITE_NAME | Bobur Gafurov |

5. Нажми **"Deploy"**
6. Подожди 2-3 минуты — сайт будет на адресе типа bobur-photo.vercel.app

## ШАГ 3 — Подключи домен boburgafurov.com

### В Vercel:
1. Зайди в свой проект → вкладка **"Settings"** → **"Domains"**
2. Введи **boburgafurov.com** и нажми **"Add"**
3. Vercel покажет тебе DNS записи — скопируй их

### В Namecheap:
1. Зайди на namecheap.com → **Domain List** → нажми **"Manage"** рядом с boburgafurov.com
2. Перейди на вкладку **"Advanced DNS"**
3. Удали все старые записи типа A и CNAME
4. Добавь записи которые показал Vercel:
   - Тип **A**, хост **@**, значение — IP от Vercel
   - Тип **CNAME**, хост **www**, значение — от Vercel
5. Нажми сохранить ✓

### Подождать 10-30 минут — и сайт будет на boburgafurov.com! 🎉

## Как добавлять фото на сайт:

1. Зайди на boburgafurov.com
2. Нажми **"+ Добавить"** в меню
3. Введи пароль (ADMIN_PASSWORD из Vercel)

5. Фото автоматически загружается на ImgBB и появляется на сайте ✅
